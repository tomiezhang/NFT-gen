# NFT-gen
## 一个NFT创作平台
---
###【开发环境】
* python3.9 + sqlite + flask
* jquery

###【实现功能】
* 简单后台系统【用户注册、登陆】
* 简单的项目创建【图片组合上传、随机合并生成】

###【to do】

* 上联盟链，随机生成作品同步到蚂蚁链上，返回txhash,代码里有接口，不过蚂蚁的API很混乱，没调通
* 上公链，结合ETH公链，自助铸造动态NFT
  * 艺术家上传不同部件
  * 在服务器上部署服务，实现访问xxx/api/ID的时候，根据概率随机合成一张图片
  * 将合成图片分发到去中心化存储IPFS
  * 将ID、NFT名字、IPFS地址等信息拼成结构化数据(json)
  * 访问xxx/api/ID的时候返回这段json
  * 在ETH上部署新合约，合约采用ERC721模版
  * 当有链上交易访问合约时，合约根据ID号去调用服务器api
  * 返回的元数据(json)写入ETH区块，完成铸造
  
###【运行】
* git clone后，通过requirements.txt来安装环境
* 直接在pycharm里点运行
* 访问http://127.0.0.1/5000 即可
